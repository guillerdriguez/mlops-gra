# Resultados del modelo baseline

Modelo: Regresión Logística

Dataset: PIMA Diabetes

Partición utilizada: 80 % entrenamiento / 20 % test

## Métricas obtenidas

| Métrica | Valor |
|--------|------|
| Accuracy | 0.7338 |
| Precision | 0.6032 |
| Recall | 0.7037 |
| F1 Score | 0.6496 |

## Observaciones

Este modelo baseline proporciona un punto de partida razonable para el problema de clasificación de diabetes.

En este contexto médico, el **recall de la clase positiva** es una métrica especialmente importante, ya que un falso negativo implica no detectar a un paciente potencialmente diabético.

El modelo será posteriormente mejorado mediante la optimización de hiperparámetros y la comparación con otros algoritmos de clasificación.
